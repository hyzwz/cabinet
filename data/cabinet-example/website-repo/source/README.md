# Weasleys' Wizard Wheezes Storefront

Small Next.js storefront for Fred and George Weasley.

## Setup

```bash
npm install
npm run dev
```

Then confirm:

- the hero feels magical and mildly unhinged
- the featured product cards render
- nobody has to debug this while holding an unstable Skiving Snackbox

## Main Files

- `package.json`
- `app/page.tsx`

## If Things Go Sideways

- `next: command not found` usually means dependencies were not installed
- a busy port means another app got there first
- if the colors feel too dramatic, that is a product decision, not a CSS accident
